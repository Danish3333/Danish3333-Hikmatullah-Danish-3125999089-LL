# Homework 001: LLM Deep Dive - Interactive Learning Website

A comprehensive, bilingual (EN/CN) single-page web application for learning Large Language Models.

## Features

- **7-Step Transformer Pipeline:** Interactive step-by-step exploration from Input to Sampling
- **Sampling Playground:** Real-time temperature, top-k, top-p, repetition penalty with probability bar chart
- **Attention Visualizer:** Interactive attention matrix with hover tooltips, editable input text
- **Mathematics Deep Dive:** 4-tab section (Overview, Formula Derivation, Worked Example, Interactive Calculator)
- **Model Architecture Comparison:** Side-by-side comparison of GPT-4, LLaMA 3, BERT, Gemini, Claude 3
- **Knowledge Quiz:** 10-question bilingual quiz with instant feedback
- **Learning Progress:** Visual progress bar tracking 7 pipeline steps

## How to Run

Open `index.html` in any web browser. No server required.
